# Comprehensive Dashboard for Enhanced Analysis of Abu Dhabi's Non-Oil Trade Data for March 2026

## Overview
This project provides a web-based dashboard to visualize and analyze the dataset titled "Non-oil Foreign Merchandise Trade Through the Ports of Abu Dhabi Emirate, March 2026". The dataset contains detailed information on trade metrics, including trade volumes, exports, re-exports, imports, and trade flows by country.

### Key Features:
- Interactive data visualization using Plotly
- Dynamic filtering and sorting options by country, HS code, and trade type
- Year-on-year growth comparison tools
- Mobile-friendly and responsive design

## Prerequisites
- Python 3.7+
- Flask: `pip install flask`
- Pandas: `pip install pandas`
- Plotly: `pip install plotly`

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repo/abu-dhabi-trade-dashboard.git
   cd abu-dhabi-trade-dashboard
   

2. Install required Python packages:
   bash
   pip install -r requirements.txt
   

3. Place the dataset file (`Non-oil_Merchandise_Trade-MAR2026.xlsx`) in the project directory.

4. Run the application:
   bash
   python app.py
   

5. Open your web browser and navigate to `http://127.0.0.1:5000/`.

## Usage
1. Open the dashboard in your browser.
2. Explore the interactive visualizations:
   - View trade metrics by country, HS code, and type (exports, imports, re-exports).
   - Compare year-on-year growth trends and trade patterns.
   - Use filters and sorting options to customize your analysis.

3. Download specific data views for further offline analysis.
